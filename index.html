Features

Unlimited Journaling

With a simple @tag, an message goes into a seperate journal. Each message can have an unlimited tags, and if a tag is used for the first time SMSJ automatically creates a new journal. An untagged message is posted in your default journal.

Integrate Everywhere

SMSJournal works on anything that can send a text message, from a flip phone to an flagship smartphone. Use Siri, Alexa, and more to journal by voice.

Secure and Private

Every message is sent directly from your phone to Google Docs by an encrypted AWS Lambda function. We do not read or log your messages, and the bits of personal data we collect to provide the service are stored in a secure AWS database.
